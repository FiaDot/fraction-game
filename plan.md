# 분수 마스터 - 10살 아이를 위한 분수 종합 게임

## Context
10살 아이가 분수를 재미있게 학습할 수 있는 단일 HTML 페이지 게임.
크기 비교, 사칙연산, 동치 분수 찾기 등 여러 유형이 섞인 종합 게임으로 구성.
iPad Air 4 가로 모드 + PC 브라우저에서 플레이.

## 파일 구조
- `/Users/fiadot/Project/fraction-game/fraction-game.html` — 단일 HTML 파일 (CSS + JS 임베디드)

## 초기 계획

### 게임 구조
1. **시작 화면**: 타이틀, 이모지 아이콘, "게임 시작" 버튼
2. **3가지 게임 모드** (랜덤 출제):
   - **크기 비교**: 두 분수를 파이차트로 시각화하고 >, <, = 선택
   - **분수 연산**: 분수 덧셈/뺄셈 문제, 답을 분자/분모 입력
   - **동치 분수 찾기**: 주어진 분수와 같은 분수를 4지선다에서 선택
3. **점수 시스템**: 맞으면 +10점, 연속 정답 보너스 +5점, 레벨업 (10문제마다)
4. **피드백**: 정답/오답 애니메이션, 격려 메시지
5. **결과 화면**: 총 점수, 정답률, 최고 연속 정답, 다시 하기 버튼

### UI/UX
- 밝고 컬러풀한 디자인 (그라디언트 배경, 둥근 카드)
- 큰 글씨, 큰 버튼 (아이 친화적)
- 파이차트는 Canvas API로 직접 그리기
- 반응형 (모바일/태블릿 대응)
- 정답 시 confetti 효과, 오답 시 흔들림 애니메이션

### 기술 구현
- 순수 HTML + CSS + Vanilla JS (외부 의존성 없음)
- Canvas 2D로 분수 파이차트 시각화
- CSS 애니메이션/트랜지션으로 인터랙션
- 분수 계산: 최대공약수(GCD) 함수로 기약분수 처리
- Web Audio API로 사운드 효과 (외부 파일 없이 코드로 생성)

## 프롬프트 이력 및 변경 사항

### 1단계: 기본 게임 생성
- **요청**: 단일 페이지로 10살 아이를 위한 분수게임 웹페이지
- **선택**: 종합 게임 (크기 비교 + 연산 + 동치 분수)
- **결과**: `fraction-game.html` 생성 (HTML + CSS + JS 올인원)

### 2단계: 사운드 추가
- **요청**: 사운드도 추가해줘
- **구현**: Web Audio API로 외부 파일 없이 사운드 생성
  - 게임 시작: 상승 멜로디 (C-E-G-C)
  - 버튼 클릭: 짧은 틱 사운드
  - 정답: 밝은 3음 상승음 (C-E-G)
  - 연속 3회+ 정답: 특별 스트릭 멜로디 (G-B-D-E)
  - 오답: 낮은 톱니파 2음
  - 결과 화면 (70%+): 팡파레 멜로디
  - 결과 화면 (70% 미만): 차분한 마무리음

### 3단계: 버튼 글자 가시성 개선
- **요청**: 보기 글자가 안보여 버튼 배경색과 글자색이 대비되도록 수정
- **문제**: 동치 분수 보기 버튼이 흰 배경 카드 위에서 흰 글씨라 안 보임
- **수정**:
  - 기본: 연보라 배경(`#f0e6ff`) + 진한 글자(`#333`) + 보라 테두리
  - 호버: 진한 보라 배경
  - 정답/오답: 초록/빨강 배경 + 흰 글자

### 4단계: 분자 동일 → 분모 동일 → 원래대로
- **요청 1**: 분수 문제에서 분자가 동일하도록 수정
- **요청 2**: (정정) 분모가 동일하게 해서 쉽게 만들려는거야
  - 크기 비교 / 연산에서 분모를 같게 하고 분자만 다르게 출제
- **요청 3**: 분모도 다르게 해줘
  - 원래대로 분자, 분모 모두 랜덤 출제로 복원

### 5단계: 팡파레 이펙트 추가
- **요청**: 문제 맞췄을 때 팡파레 이펙트 추가
- **구현**:
  - 화면 금색 플래시
  - 중앙에서 퍼지는 컬러 링 3개
  - 12방향으로 터지는 별/이모지 (⭐🌟✨💫🎉🎊💥🔥)
  - 기존 confetti + 사운드와 함께 재생

### 6단계: iPad Air 최적화
- **요청**: 아이패드 에어에서 화면 80% 차도록
- **구현**:
  - 컨테이너 `80vw` 너비, `80vh` 최소 높이
  - 태블릿(700px+) 미디어쿼리: 글씨, 버튼, 파이차트 등 확대
  - 파이차트 Canvas: 태블릿에서 160px

### 7단계: iPad Air 4 가로 모드 + PC 대응
- **요청**: 아이패드 에어4 가로로 플레이, PC에서도 잘 보이게
- **구현**: iPad Air 4 가로(1180x820) 해상도 기준
  - 3단계 미디어쿼리:
    - 태블릿 & PC (700px+): 큰 요소
    - iPad 가로 (700px+ & 높이 900px 이하): 조밀한 레이아웃
    - 모바일 (500px 이하): 소형 레이아웃
  - `overflow-y: auto`로 스크롤 허용
  - Canvas 크기: 가로 모드 110px, 세로/PC 140px, 모바일 90px

### 8단계: 분수 연산 +/- 스피너 버튼
- **요청**: 분수 계산 문제에서 분자와 분모 양옆에 +, - 버튼 추가
- **구현**:
  - 분자/분모 입력 양옆에 원형 −/+ 버튼 배치
  - 터치(iPad)에서 직관적으로 숫자 변경 가능
  - 분자 0 이상, 분모 1 이상 제한
  - 브라우저 기본 숫자 스피너 숨김
  - 기존 직접 입력 + Enter 제출도 유지
